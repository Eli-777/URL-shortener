
# restaurant list
用Node.js、Express、handlebars練習的作品，內容主要為縮短網址。

## 安裝方法
下載專案
```
git clone https://github.com/Eli-777/URL-shortener.git
```
安裝 Package
```
npm install
```
使用 nodemon 啟動伺服器
```
npm run dev
```
或正常啟動
```
npm start
```

## 功能 
- 在首頁輸入網址，點擊按鈕可產生短網址。
- 產生短網址後，可按按Copy鈕複製連結。


## 使用工具
- Node.js v12.16.2 -執行環境
- Express V4.17.1 -框架
- Express-handlebars V4.0.4 -模板引擎
- body-parser: 1.19.0
- mongoDB Community Server: 4.2.6
- mongoose: 5.9.13
